# indiAccounting

Privacy-first desktop accounting and payroll for freelancers, contractors, and small businesses. 
Your books live on your own machine — not in the cloud — and work offline for day-to-day bookkeeping. 
One-time purchase, no subscription.

## Installation
```bash
flatpak install flathub com.indomitusgroup.indiAccounting
```

## Features

- Invoicing & Payments
- Full Payroll for All 50 States
- Banking & Reconciliation
- Inventory
- Services
- Time Tracking
- Mileage Tracking
- Financial Reports
